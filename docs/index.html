プロジェクト概要

PersonaAgent Factory は、大規模製造業向けに10種類の専門ペルソナを提供する本格的なAIコンサルテーションシステムです。

現在本番環境で稼働中で、工場作業員から管理職、営業担当まで幅広い職種の視点で業務課題を分析・解決支援します。

10種類の製造業ペルソナ
マルチAIプロバイダー対応
チャット履歴管理・CSV出力
2段階Enter確認システム
リアルタイムログ監視
自動デプロイメント
